🛡️ MiniVault v2.9 | Encryption Engine

MiniVault is a high-performance, standalone security tool engineered for mission-critical tasks.

💎 Technical Excellence

🔑 Cryptographic Core

  • Argon2id Key Derivation: Configured with a 1.0 GB Memory Cost. Resistance against brute-force and TMTO exploits.
  • AES-256 GCM: Military-grade authenticated encryption.
  • Hardware Acceleration: Native support for Intel® AES-NI.

🧠 Advanced Memory Management

  • Memory-Hard Security: 1 GB RAM lock during transformation sandbox.
  • Clipboard Sanitization: Real-time purging of Windows Clipboard & Win+V.

💻 System Requirements

  • Processor: Intel Core i5-6200U (Min) / i7-7560U (Rec)
  • Memory: 8 GB LPDDR3 (Min) / 16 GB (Rec)
  • OS: Windows 10/11 64-bit

📜 Legal & Compliance

Privacy First: Zero telemetry. Zero cloud-sync. Your keys never leave your local RAM.
